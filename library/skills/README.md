# Harness config skill library

This directory contains public, customer-facing skills that help people adopt
Harness config in their own repositories.

These skills are separate from the repository's internal development skills in
`.harness/resources/skills`. Internal skills support development of this
project; library skills are installable guidance for users.

## Skills

- `harness-config`: Set up, migrate, validate, activate, and troubleshoot
  Harness config in a repository, including portable resource groups,
  profiles, local layers, and generated harness surfaces.

Install with a skill installer from:

```text
https://github.com/reachjalil/harness-config/tree/main/library/skills/harness-config
```
